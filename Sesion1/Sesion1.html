det(A)

A.^2

A(1,:) % Coge la primera fila todos los elemento de la columa

Funcines amb item "function" (ttienen que ir al final del fichero)
function [ea,er,ep] = errorfun(x,y)
%caclula l'errror absolut, l'error relatiu i l'error en %
ea = abs(x-y);
er = ea/abs(x);
ep = er*100;
end
Ejecucion de una funcion
[ea,er,ep] = errorfun(pi,22/7)
Function definitions in a script must appear at the end of the file.
Move all statements after the "errorfun" function definition to before the first local function definition.
